# project-murgia-noir: Cinematic Homepage

## 🎯 Goal
Create an immersive, high-fidelity homepage for **Murgia Liquori** using the **"Option A: Cinematic Narrative"** structure. The design will follow a high-contrast **"Golden Noir"** aesthetic with sharp geometric precision.

---

## 🎨 Design Commitment: "Murgia Golden Noir"

- **Topological Choice:** **Single Stream Fragmentation**. Overlapping layers that break standard vertical/horizontal grid logic.
- **Typography:** 
  - Headings: `Bebas Neue` (Condensed, All-Caps, Massive)
  - Body: `Montserrat` (Geometric, Wide spacing)
- **Palette (NO PURPLE):** 
  - Primary: `#F4B400` (Villacidro Yellow)
  - Background: `#040404` (Textured Noir)
- **Risk Factor:** Intentionally overlapping text and image fragments to create a sense of depth and artisanal "collage".
- **Cliché Liquidation:** Killed the "Safe Split" (Left text/Right image) and Bento grids.
